# 🛡️ SentinelSphere — AI-Powered Home Security Dashboard

An AI-powered home security analytics dashboard that simulates real-time threat detection, digital twin visualization, and intelligent security monitoring. Built with **Python**, **Streamlit**, and **Plotly**, SentinelSphere provides an interactive dashboard for analyzing home security events, monitoring activity, and generating AI-driven insights.

---

## ✨ Features

* 🏠 Interactive 3D Digital Twin of a smart home
* 🚨 Real-time security alerts with threat classification
* 📊 Activity heatmaps and security analytics
* 📈 Timeline replay of security events
* 🤖 AI-generated security insights and risk predictions
* ⚙️ Custom daily routine configuration
* 🌍 Community threat heatmap
* 📱 Interactive verification and alert management

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Amank0704/SentinelSphere.git
cd SentinelSphere
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
